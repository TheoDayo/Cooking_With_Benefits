Instructions on how to run the program:

1) Install xampp.
2) Download the program folder into C:\xampp\htdocs folder.
3) Start Apache and MySQL in xampp
4) Open PHPMyAdmin 
5) Import cwbdb_creation.sql and cwbdb_inserttestdata.sql from C:\xampp\htdocs\Cooking_With_Benefits\dbscripts 
to MySQL (THIS CREATES A TABLES AND INSERTS SAMPLE DATA)
6) In a web browser, enter http://localhost/Cooking_With_Benefits/
7) Click Connect or Log In button
8) Click "Create an an account!"
9) Enter values until finished.
10) repeat step 7
11) Log In.
12) Click "Search Recipe" in the top nav bar.
13) Type "s" and results should appear.
